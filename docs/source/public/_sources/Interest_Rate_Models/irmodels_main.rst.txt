Interest Rate Models 
=======================

Interest rate models are mathematical frameworks used to describe and analyze the behavior and dynamics of interest rates. 
These models help financial professionals, economists, and researchers understand and predict interest rate movements, which are crucial in pricing and managing interest rate products, valuing fixed income securities, and making informed investment decisions.
The code and documentation for these products can be found at this location:

- `Documentation <Interest_Rate_Models_Documentation.html>`_
- `Code <ir_models.html>`_
