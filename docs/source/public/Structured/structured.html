run_mc(mc_engine, dates, discount_curve, no_sims, statistics_gatherer=None)#

Run a Monte Carlo simulation for a given instrument.

If the instrument has an underlying, the Monte Carlo engine’s price method is used. Otherwise, the instrument’s get_px method is called.

Parameters:

  • mc_engine (object) – The Monte Carlo engine to use for pricing.

  • dates (list or pandas.DatetimeIndex) – The dates for which to calculate prices.

  • discount_curve (object) – The discount curve to use for pricing.

  • no_sims (int) – The number of simulations to run.

  • statistics_gatherer (object) – An optional statistics gatherer. If provided, will be passed to the Monte Carlo engine.

Returns:

The simulated prices for the instrument.

Return type:

float or np.ndarray

get_px(dates, discount_curve)#

Compute the price of an option.

Parameters:

  • dates (array-like of pandas.Timestamp) – Dates at which to compute the forward price.

  • discount_curve (pricing.discount_curves.DiscountCurve, optional) – Provides the discounts for the computation of the strikes if these are not given. Default is None.

Returns:

  • For LognormalAsset, the function returns a tuple containing arrays of forward prices, d1, d2, discount factors, and volatilities.

  • For NormalAsset, the function returns a tuple containing arrays of zero rates, volatilities, and d values.

Return type:

The function’s output(a tuple) varies depending on the type of asset

get_gamma(dates, discount_curve)#

Calculate the gamma of an option. Gamma is the second derivative of the option’s price with respect to the underlying asset’s price.

Parameters:

  • dates (array-like of pandas.Timestamp) – Dates at which to compute the gamma.

  • discount_curve (pricing.discount_curves.DiscountCurve) – Provides the discounts for the computation of the gamma.

Returns:

gamma – Returns gamma for the given dates if the underlying asset is of type LognormalAsset. For a basket of assets or other unsupported types, a message is printed and None is returned.

Return type:

numpy.ndarray or None

Notes

This method is not applicable for a basket of assets; in such cases, Monte Carlo methods are suggested. If the underlying asset is not of type LognormalAsset, the method also returns None, indicating unsupported asset dynamics.
