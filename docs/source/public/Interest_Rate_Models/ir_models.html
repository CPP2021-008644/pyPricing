ir_models.set_past_dates(arr, a, dates1, dates2)#

Returns a modified array where the value ‘a’ is set at positions where dates1 < dates2.

Parameters:

  • arr (np.ndarray) – The original array to be modified.

  • a (float) – The value to be set in the array at the specified positions.

  • dates1 (pandas.DatetimeIndex) – First set of dates.

  • dates2 (pandas.DatetimeIndex) – Second set of dates.

Returns:

The modified array where at positions dictated by dates1 being less than dates2, the value equals ‘a’.

Return type:

np.ndarray

Examples

>>> dates1 = pd.DatetimeIndex(["20211229", "20221231", "20231231"])
>>> dates2 = pd.DatetimeIndex(["20211229", "20211230"])
>>> arr = np.full((dates1.size, dates2.size), 1)
>>> set_past_dates(arr, 0, dates1, dates2)
np.array([[1, 0], [1, 1], [1, 1]])
